# Lab Setup

This lab was built in a controlled virtual environment for learning and defensive security research.

## Environment
- Windows 10 virtual machine
- Local user accounts for testing
- Windows Security event logging enabled
- Sysmon (optional)

## Tools Used
- Windows Event Viewer
- Splunk (optional)
- Kali Linux (attack simulation)
- PowerShell

## Logging Sources
- Security Event Log
- System Event Log
- Application Event Log
- Sysmon logs (if installed)

## Notes
All activity was generated in a private lab environment. No production systems were involved.
