# Sample Logs

This folder contains sample log extracts collected from the lab environment.

## Log Types
- Windows Security Event Log
- System Event Log
- Application Event Log
- Sysmon logs (if enabled)

## Data Handling
- All usernames, hostnames and IP addresses have been anonymised.
- Logs are limited to short time windows for demonstration purposes.

## Purpose
These logs are used to test detection rules and investigation workflows.
